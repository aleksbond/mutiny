mutiny
=====
generic, reusable HTML5 elements

Getting Started
-----
    $ git clone https://git.cashnetusa.com/ui/mutiny.git
    $ cd mutiny
    $ bundle install
    $ script/rails server
    $ open http://localhost:3000/
